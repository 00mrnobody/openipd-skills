## 1. 项目背景与机会

### 1.1 市场机遇

中国AI情感陪伴市场正处于爆发式增长期：

| 市场指标 | 数据 | 来源 |
|---------|------|------|
| 当前市场规模 | 38.66亿元 | MKTPDT市场分析 |
| 年复合增长率(CAGR) | 148% | MKTPDT市场分析 |
| 增长驱动因素 | 独居人群增加、AI技术成熟、情感陪伴需求上升 | 综合分析 |

**市场窗口期判断**：当前是进入AI硬件情感陪伴市场的战略窗口期，竞争对手尚未形成垄断格局，技术成熟度已达到商业化临界点。

### 1.2 客户痛点

目标用户群体面临的核心痛点：

1. **独居人群的孤独感**：中国独居人口超过1.25亿，情感陪伴需求强烈但缺乏合适解决方案
2. **现有产品体验差**：智能音箱只能被动响应，缺乏情感交互能力
3. **云端AI+本地硬件割裂**：现有方案要么依赖手机App，要么本地算力不足导致AI体验受限
4. **移动能力缺失**：现有桌面AI产品固定放置，无法主动跟随和陪伴

### 1.3 战略契合度

本项目符合公司战略方向：
- ✅ 进军AI硬件赛道，抢占情感陪伴市场入口
- ✅ 云端AI+IoT硬件的生态协同
- ✅ 可扩展的软硬件平台，支持后续产品线延伸

### 4.2 技术路线

**AI架构**：纯云端AI（无本地NPU）
- 优势：算力无限扩展、模型持续升级、硬件成本可控
- 风险：网络依赖、延迟控制

**技术栈**：
```
云端：大模型API + 情感分析引擎 + 对话管理系统
设备端：Linux嵌入式 + 音视频采集 + 底盘控制（移动版）
App端：iOS/Android原生应用
```

### 4.3 双版本差异化实现

| 差异化点 | 实现方案 |
|---------|---------|
| 底盘差异 | 固定版取消移动底盘、电机、电池，使用相同主控板 |
| BOM差异 | 固定版BOM降低$30-40（约¥200-280） |
| 软件差异 | 移动版增加底盘控制、路径规划、防跌落算法 |
| 结构差异 | 共享70%以上结构件，仅底座/外壳不同 |

### 10.2 风险应对详细方案

#### R-01: 移动底盘成本风险

**触发条件**：移动底盘实际成本>$40
**应对措施**：
1. 启动双源采购，确保至少两家合格供应商
2. 若成本无法降低，固定版作为主销产品，移动版限量销售
3. 评估底盘方案简化可行性

#### R-02: 云成本失控风险

**触发条件**：单用户年云成本>¥1,500
**应对措施**：
1. 建立实时用量监控和预警机制
2. 与云服务商谈判阶梯定价
3. 推出云服务订阅包，转移成本给用户
4. 高频用户引导至付费套餐

#### R-04: 防跌落安全风险

**触发条件**：防跌落测试通过率<99.9%
**应对措施**：
1. 传感器双重冗余设计
2. 强制物理护栏设计（不可拆卸）
3. 跌落自动断电保护
4. 产品责任险覆盖
