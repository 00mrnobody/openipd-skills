# OpenClaw 桌面精灵示例 Demo

这是 OpenIPD Skill 的一个端到端示例产出，展示了从概念到计划阶段的核心交付物。

## 文件清单

- `01-charter.md`：项目 Charter（概念阶段核心输入）
- `02-market-analysis.md`：市场分析（MKTPDT）
- `03-tech-feasibility-analysis.md`：技术可行性分析（SE/HW方向）
- `04-financial-feasibility.md`：财务可行性分析（FPDT）
- `05-mfg-procurement-plan.md`：制造与采购计划（MNFPDT/PROPDT）
- `06-pdcp-resolution.md`：PDCP 决议
- `07-ipd-exec-summary.md`：执行摘要

## 审计说明

- 本 Demo 仅包含 Markdown 交付物，不含密钥、账号配置、私有凭据文件。
- 该示例用于展示 OpenIPD 协同流程与文档产出结构。
